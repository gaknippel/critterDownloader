# critterDownloader

a simple YouTube video and audio downloader.

<div align="center">
  <img src="ytdlpAD.gif" alt="Critter Logo" />
</div>

## features

- downloading video & audio in the BEST quality with no dumb subscriptions
- simple ui
- no bloated ads
- ability to save to path
- changeable theme

## installation

download the latest release and run the installer.

https://github.com/gaknippel/critterDownloader/releases

## usage

1. paste a YouTube link
2. choose video or audio
3. click download! easy!

## built with

- tauri
- react
- yt-dlp
- shadcnui
- reactbits
